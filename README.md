# ⚡ Promptly AI

**Promptly AI** is a full-stack AI platform that lets users **chat, generate images, and extract insights from PDFs** — all in one place.

---

## 🚀 What It Does

* 💬 **AI Chat** – Smart conversational assistant
* 🖼️ **Image Generation** – Create images from text prompts
* 🌐 **Web Search** – Get real-time information from the internet
* 📄 **PDF Intelligence** – Upload PDFs and ask questions from them

---

## 🧠 Core AI Features

* Document **upload → chunking → embedding → vector database**
* Semantic search using **vector similarity**
* Context-aware responses from uploaded data
* Multi-modal AI (text + image)

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express
* **Database:** MongoDB
* **AI/ML:** Embeddings + Vector DB + LLM APIs

---

## ⚡ Why It's Impressive

* Combines **ChatGPT + Image AI + Document AI** in one system
* Implements **real-world AI pipeline (RAG architecture)**
* Handles **document understanding efficiently**
* Scalable backend with clean API structure

* Smart chatbot applications
